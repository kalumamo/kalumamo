# Ahadu Bank AI-Powered Digital Banking Product Evaluation Platform

## Overview
Enterprise-grade AI-powered platform for evaluating digital banking product performance.

## Architecture
- **Backend**: Python 3.12 + FastAPI + SQLAlchemy + Celery + Redis
- **Database**: MySQL
- **ML**: Scikit-Learn (Logistic Regression, Ridge Regression, KNN)
- **Frontend**: Next.js 15 + TypeScript + TailwindCSS + Shadcn UI
- **Infrastructure**: Docker + Docker Compose + Nginx + GitHub Actions

## Quick Start
```bash
docker-compose up --build
```

## Services
- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- API Docs: http://localhost:8000/docs
- Redis: localhost:6379
- MySQL: localhost:3306
