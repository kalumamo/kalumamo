================================================================================
TROUBLESHOOTING
================================================================================

Problem: Dashboard shows "No connection"
Solution: 
  1. Check backend is running on 8000
  2. Check frontend .env.local points to localhost:8000
  3. Restart frontend: npm run dev

Problem: Scores not changing after upload
Solution:
  1. Check backend logs for "Scored X products"
  2. Hard refresh dashboard (Ctrl+Shift+R)
  3. Check database: SELECT COUNT(*) FROM scores;

Problem: Backend won't start
Solution:
  1. Clear __pycache__ directories
  2. Check Python version (3.10+)
  3. Install requirements: pip install -r requirements.txt
  4. Try: python -m uvicorn app.main:app --port 8000

Problem: Upload fails
Solution:
  1. Check CSV format (product_code, period_date required)
  2. Check product codes exist in database
  3. Check date format (YYYY-MM-DD)
  4. See backend logs for error details
