================================================================================
                    TROUBLESHOOTING
================================================================================

Problem: Scores don't change after upload
Solution: 
  • Refresh page (Ctrl+R)
  • Check backend is still running
  • Check backend logs for errors
  • Try uploading next dataset

Problem: Old data still showing
Solution:
  • Wait 30 seconds
  • Go to Settings and click "Run Feature Engineering"
  • Refresh all pages
  • Check database

Problem: Previous scores not shown
Solution:
  • This is OK for first upload
  • Should show "Previous: 90" starting with 2nd upload
  • If not shown, reload page

Problem: Alerts don't update
Solution:
  • Wait 30 seconds
  • Refresh page
  • Check database for new alerts

Problem: Recommendations same as before
Solution:
  • May take 60 seconds to generate
  • Refresh page after waiting
  • Check backend for processing status
